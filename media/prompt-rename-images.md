# Image Rename Prompt

Paste this into a Cowork session after selecting a folder containing your images:

---

Look through all image files in my folder whose filenames contain "IMG" (case-insensitive). For each one:

1. Open and visually review the image to understand what the photograph depicts.
2. Rename the file with a short, descriptive, lowercase, hyphen-separated name that captures the main subject of the photo (e.g., "sunset-over-mountain-lake.jpg", "group-at-picnic-table.jpg", "storefront-main-street.jpg").
3. Keep the original file extension.
4. Make sure no two files end up with the same name — if there are duplicates, add a number suffix (e.g., "sunset-over-lake-2.jpg").
5. After renaming, show me a summary table with the old filename and new filename for each image.

Do not rename any files that already have descriptive names.
