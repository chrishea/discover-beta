Annotation legend

Each numbered pin points at a standardized decision drawn from the audit. The tag at the end of each entry names the dimension it addresses.

  1. Hero badge. Transparent white-on-dark pill — rgba(255,255,255,0.14) background, 1px white-30% border, uppercase 0.8rem text. Consistent across MAIN/SHOP/STAY/CAMP/HIKE; DO and EAT override to unique colors (transparent white / light green) — treat those as outliers. Color
  2. H1 — Hero title. IBM Plex Serif 700, clamp(2.5rem, 5vw, 4rem), white, line-height 1.1. Matches common.css default. Each word wrapped in a .word span so js/common.js can stagger-fade them in. Typography
  3. Info bar — 4 items. Four is the most common count across SHOP/CAMP/HIKE. Black background, white text, small circle icons in 10% white, 0.85rem body. Drop emoji icons or keep them — choose one (majority is "keep", 5-to-2). Structure
  4. TOC pill nav. Five pills, .section-alt-colored strip, pill shape (border-radius 999px), 1px black border, white fill. Present on MAIN/SHOP/DO/STAY/EAT. CAMP and HIKE omit — document the exemption if kept. Structure
  5. H2 — Section title. IBM Plex Serif 700, clamp(1.8rem, 3vw, 2.5rem), pure black (#000). Fixes MAIN's broken rule at line 364 (unclosed single quote in font-family). Typography
  6. Intro sidebar (Quick Facts). Optional right-rail block inside the Overview section. 1px black border, white fill, H3 in Plex Serif. Replaces the ad-hoc "market-grid" pattern used inconsistently today. Structure
  7. Alternating section backgrounds. .section (white) and .section-alt (#f2f2f2). Six of seven files use this; MAIN uses .guide-section numbered pills instead — convert MAIN. Structure
  8. Card — .lodge-card. 1px solid black border, 16:9 image area, body padding 1.1rem, H3 in Plex Serif 1.15rem, 0.95rem body. Cards with images are the 4-of-7 majority (DO/STAY/EAT/HIKE); image-less variants still use the same shell. Visual elements
  9. Compare table. Black header row, uppercase white labels 0.8rem, alternating row stripe #fafafa, 1px dividers. Codifies the pattern seen in STAY, DO, EAT, SHOP. Structure
  10. Tip box callout. 4px solid-black left border, white background, Plex Serif H4 in uppercase. Single callout style — replaces STAY's .pros-cons + CAMP's .tip-card + HIKE's informal inline tips. Visual elements
  11. Last-updated stamp + Sources. New surface area. Neither exists on any of the seven audited files today. Black pill for the date, a topped-rule block with a numbered <ol> below. Adds credibility at minimal cost. Tone / formatting
  12. CTA section. Solid black background, centered Plex Serif H2, white copy, one primary button. Matches the pattern across all seven files. Structure
